FUNÇÃO 1 — submitTemplate
Envia um novo template para aprovação da Meta.

export async function submitTemplate(params: {
  phoneNumberId: string;
  accessToken: string;
  wabaId: string;
  templateName: string;
  category: 'MARKETING' | 'UTILITY' | 'AUTHENTICATION';
  language: string;
  headerType?: 'TEXT' | 'IMAGE' | 'DOCUMENT' | 'VIDEO' | null;
  headerContent?: string;
  bodyText: string;
  footerText?: string;
  buttons?: Array<{
    type: 'QUICK_REPLY' | 'URL' | 'PHONE_NUMBER';
    text: string;
    url?: string;
    phone?: string;
  }>;
}): Promise<{ templateId: string; status: string }>

Lógica:
1. Montar o payload para POST https://graph.facebook.com/v19.0/{wabaId}/message_templates
2. O payload tem: name, category, language, components
3. components é um array. Montar assim:
   - Se headerType existe: { type: "HEADER", format: headerType, text: headerContent }
   - Sempre: { type: "BODY", text: bodyText }
   - Se footerText: { type: "FOOTER", text: footerText }
   - Se buttons.length > 0: { type: "BUTTONS", buttons: [...] }
4. Fazer fetch com Authorization: Bearer {accessToken}
5. Se resposta ok: retornar { templateId: data.id, status: data.status || 'PENDING' }
6. Se erro: lançar Error com a mensagem da Meta (data.error?.message)

FUNÇÃO 4 — countTemplateVariables
Conta quantas variáveis {{N}} existem em um texto.

export function countTemplateVariables(text: string): number {
  const matches = text.match(/\{\{\d+\}\}/g);
  return matches ? matches.length : 0;
}

--- ARQUIVO 2: Adicionar rotas em server/routes/whatsapp-official.ts ---

No router de whatsapp-official (já existente), adicione as seguintes rotas:

GET /templates
- Buscar whatsapp_official_connections WHERE workspace_id = workspaceId
- Se não existe conexão: retornar []
- Buscar todos os templates da tabela whatsapp_message_templates
  WHERE workspace_id = workspaceId ORDER BY created_at DESC
- Retornar array de templates

POST /templates
- Body: { templateName, category, language, headerType, headerContent,
          bodyText, footerText, buttons }
- Validações:
  * templateName obrigatório, só letras minúsculas, números e underscores
    (regex: /^[a-z0-9_]+$/ — a Meta exige esse formato)
  * bodyText obrigatório, máximo 1024 chars
  * category deve ser: MARKETING, UTILITY ou AUTHENTICATION
- Buscar conexão Meta ativa do workspace
- Chamar submitTemplate do serviço acima
- Inserir na tabela whatsapp_message_templates com status 'PENDING'
- Retornar o template criado com status 201

POST /templates/:id/sync
- Buscar o template pelo id e workspace_id
- Buscar a conexão Meta ativa
- Chamar syncTemplateStatus
- Se retornou dados: UPDATE whatsapp_message_templates
  SET status = novoStatus, rejection_reason = motivo, updated_at = now()
- Retornar o template atualizado

POST /templates/sync-all
- Buscar todos os templates PENDING ou em revisão do workspace
- Para cada um: chamar syncTemplateStatus e atualizar no banco
- Retornar { updated: N, total: M }

DELETE /templates/:id
- Buscar o template pelo id e workspace_id
- Se status for APPROVED, fazer DELETE na Meta API:
  DELETE https://graph.facebook.com/v19.0/{wabaId}/message_templates
  ?name={templateName}
  Authorization: Bearer {accessToken}
- Deletar da tabela whatsapp_message_templates
- Retornar { success: true }

--- ARQUIVO 3: Scheduler de sincronização automática ---

No server/index.ts, dentro do bloco do setInterval existente (após startISPSchedulers),
adicione um novo scheduler que roda a cada 30 minutos:

setInterval(async () => {
  try {
    const { db: schedDb } = await import("./db");
    const { whatsappOfficialConnections, whatsappMessageTemplates } = await import("@shared/schema");
    const { eq, inArray } = await import("drizzle-orm");
    const { syncTemplateStatus } = await import("./services/meta-whatsapp-templates");

    const pendingTemplates = await schedDb
      .select()
      .from(whatsappMessageTemplates)
      .where(inArray(whatsappMessageTemplates.status, ['PENDING', 'IN_APPEAL']));

    if (pendingTemplates.length === 0) return;

    for (const tmpl of pendingTemplates) {
      try {
        const [conn] = await schedDb
          .select()
          .from(whatsappOfficialConnections)
          .where(eq(whatsappOfficialConnections.workspaceId, tmpl.workspaceId))
          .limit(1);

        if (!conn) continue;

        const result = await syncTemplateStatus({
          wabaId: conn.wabaId,
          accessToken: conn.accessToken,
          templateName: tmpl.templateName,
          language: tmpl.language,
        });

        if (result && result.status !== tmpl.status) {
          await schedDb
            .update(whatsappMessageTemplates)
            .set({
              status: result.status,
              rejectionReason: result.rejectionReason || null,
              approvedAt: result.status === 'APPROVED' ? new Date() : null,
              updatedAt: new Date(),
            })
            .where(eq(whatsappMessageTemplates.id, tmpl.id));

          console.log(`[HSM Sync] Template "${tmpl.templateName}" status: ${tmpl.status} → ${result.status}`);
        }
      } catch (e: any) {
        console.error(`[HSM Sync] Error syncing template ${tmpl.id}:`, e.message);
      }
    }
  } catch (e: any) {
    console.error("[HSM Sync] Scheduler error:", e.message);
  }
}, 30 * 60 * 1000);
