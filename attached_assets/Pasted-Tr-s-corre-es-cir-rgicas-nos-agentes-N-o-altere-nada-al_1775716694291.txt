══ GAP 2 — F8: trocar escalação via prompt por redirectTo ══

Em server/services/agents/financeiroAgent.ts, no método
handle(), localize onde situationCode === 'F8' é tratado
(ou onde a IA é chamada sem escalação para F8).

Adicione o bloco abaixo junto com F7, F9, F10
na verificação de escalações obrigatórias:

  if (situationCode === 'F8') {
    // Coletar descrição antes de escalar
    const descricao =
      input.textoEfetivo ??
      dados.descricao_contestacao ??
      'não informada';

    // Salvar para o handoff summary
    dados.descricao_contestacao = descricao;

    return {
      message: '',
      redirectTo: 'HUMANO',
      guardFeedback:
        `Contestação de cobrança. Descrição: ${descricao}. ` +
        `Sem autoridade para estornar ou cancelar cobranças — ` +
        `requer análise do financeiro humano.`,
    };
  }

Isso garante que F8 aparece nos logs de GUARD_REDIRECT
e no handoff summary igual aos demais.
