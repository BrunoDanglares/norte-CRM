## FIX 2: Fix N+1 query in user listing (A4)

File: server/routes.ts lines ~2214-2219

PROBLEM:
```typescript
// N+1 — one query per team:
for (const t of wsTeams) {
  const members = await storage.getTeamMembers(t.id); // query per iteration!
  ...
}
```

SOLUTION: Fetch all members in one query:
```typescript
// Get all team IDs at once
const teamIds = wsTeams.map(t => t.id);

// Single query for all members
const allMembers = await db.select()
  .from(teamMembers)
  .where(inArray(teamMembers.teamId, teamIds));

// Group by teamId in memory
const membersByTeam = allMembers.reduce((acc, member) => {
  if (!acc[member.teamId]) acc[member.teamId] = [];
  acc[member.teamId].push(member);
  return acc;
}, {} as Record<number, typeof allMembers>);

// Use the grouped result
const result = wsTeams.map(t => ({
  ...t,
  members: membersByTeam[t.id] || []
}));
```

Import inArray from drizzle-orm if not already imported.
