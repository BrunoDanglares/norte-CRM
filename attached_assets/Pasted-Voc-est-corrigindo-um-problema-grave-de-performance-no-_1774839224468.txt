PROBLEMA: Em server/services/whatsappWebJS.ts por volta da linha 928, o código faz storage.getConversations(workspaceId) para carregar TODAS as conversas na memória e depois usa .find() para localizar pelo telefone. Com milhares de conversas isso é ineficiente e pode causar timeout.

TAREFA: Criar método dedicado getConversationByPhone e substituir o padrão de .find().

PARTE 1 — Adicionar método getConversationByPhone em server/storage.ts:

```typescript
async getConversationByPhone(
  workspaceId: string,
  phone: string
): Promise {
  const result = await db
    .select()
    .from(conversations)
    .where(
      and(
        eq(conversations.workspaceId, workspaceId),
        eq(conversations.telefone, phone)
      )
    )
    .limit(1);

  return result[0];
}
```
