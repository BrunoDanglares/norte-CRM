## OBJETIVO
Implementar endpoint POST /api/n8n/send-list no ChatBanana
para envio de mensagens interativas do tipo lista no WhatsApp.

## PASSO 1 — CRIAR ENDPOINT em server/routes/n8n.ts

router.post('/send-list', requireAuth, async (req, res) => {
  const {
    phone,
    workspaceId,
    conversationId,
    texto,
    footer,
    button_label,
    sections
  } = req.body;

  if (!phone || !workspaceId || !texto) {
    return res.status(400).json({ error: 'phone, workspaceId e texto são obrigatórios' });
  }

  try {
    // Se sections vier como string (N8N serializa assim), parsear
    const parsedSections = typeof sections === 'string'
      ? JSON.parse(sections)
      : sections || [];

    // Se não houver opções, envia texto simples como fallback
    if (!parsedSections.length || !parsedSections[0]?.rows?.length) {
      await sendWhatsAppTextMessage({ to: phone, text: texto, workspaceId });
      return res.json({ success: true, type: 'text_fallback' });
    }

    // Envia lista interativa
    await sendWhatsAppListMessage({
      to: phone,
      workspaceId,
      body: texto,
      footer: footer || 'Escolha uma opção ou digite livremente',
      buttonLabel: button_label || 'Ver opções',
      sections: parsedSections
    });

    // Salva no histórico da conversa se conversationId fornecido
    if (conversationId) {
      await db.insert(messages).values({
        conversationId,
        workspaceId,
        content: texto,
        type: 'list_message',
        direction: 'outbound',
        metadata: { sections: parsedSections },
        createdAt: new Date()
      });
    }

    return res.json({ success: true, type: 'list_message' });

  } catch (err) {
    console.error('[send-list] erro:', err);
    // Fallback para texto simples em caso de erro
    try {
      await sendWhatsAppTextMessage({ to: phone, text: texto, workspaceId });
      return res.json({ success: true, type: 'text_fallback_error' });
    } catch (fallbackErr) {
      return res.status(500).json({ error: 'Falha ao enviar mensagem' });
    }
  }
});

## PASSO 2 — IMPLEMENTAR sendWhatsAppListMessage
No arquivo do Meta WhatsApp service (metaWhatsappService.ts ou similar),
adicionar a função:

async function sendWhatsAppListMessage({
  to, workspaceId, body, footer, buttonLabel, sections
}) {
  // Buscar credenciais do workspace
  const phoneNumberId = await getPhoneNumberId(workspaceId);
  const accessToken = await getAccessToken(workspaceId);

  const payload = {
    messaging_product: 'whatsapp',
    to,
    type: 'interactive',
    interactive: {
      type: 'list',
      body: { text: body },
      footer: { text: footer },
      action: {
        button: buttonLabel,
        sections
      }
    }
  };

  const response = await fetch(
    `https://graph.facebook.com/v18.0/${phoneNumberId}/messages`,
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${accessToken}`,
        'Content-Type': 'application/json; charset=utf-8'
      },
      body: JSON.stringify(payload)
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(`Meta API error: ${JSON.stringify(error)}`);
  }

  return response.json();
}
