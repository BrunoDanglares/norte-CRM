BLOCO 16 — GERAR DOCUMENTO (PDF / CONTRATO)
🎯 Objetivo

Gerar documentos dinâmicos com dados do lead (contrato, proposta, orçamento).

🧠 PROMPT BASE (ARQUITETURA)
Crie um bloco chamado "Gerar Documento".

Esse bloco deve:

1. Permitir escolher um template (HTML ou texto estruturado)
2. Substituir variáveis dinâmicas:
   {{nome}}, {{cpf}}, {{valor}}, {{data}}, etc

3. Gerar um arquivo PDF automaticamente

4. Salvar o documento no sistema (storage interno ou S3)

5. Retornar:
   - URL do documento
   - ID do documento

6. Permitir ações seguintes:
   - enviar via WhatsApp
   - enviar por email
   - salvar no CRM

7. Permitir múltiplos templates reutilizáveis
🔧 PROMPT FRONT-END
Crie interface do bloco "Gerar Documento":

Campos:
- Nome do documento
- Selecionar template
- Editor de template (WYSIWYG ou HTML)
- Variáveis disponíveis (sidebar)
- Botão: "Gerar preview"

Extras:
- botão "duplicar template"
- botão "testar com dados fictícios"
⚙️ PROMPT BACK-END
Implemente:

1. Motor de renderização (Handlebars ou similar)
2. Conversão HTML → PDF (puppeteer ou pdfkit)
3. Upload do PDF para storage
4. Retorno:
{
  url: "...",
  document_id: "...",
  size: "...",
  created_at: "..."
}
